Expense Tracker

A beginner-friendly Python project that demonstrates how to track daily expenses using JSON files and dictionaries.

📌 Description

This project allows users to:

Add expenses
View all expenses
Calculate total expenses
Save data permanently
Load data automatically
Exit safely

All expense data is stored in Expensetracker.json, so records remain available even after closing the program.
